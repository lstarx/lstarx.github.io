    Ubuntu安装Sublime并配置python开发环境.md

    Lstarx
    2018-12-10

    前言：

    第一次用Git+hexo搭建博客，
    markdown来编辑，由于markdown不怎么熟练，
    排版不是很好，内容还是有的，大家多多包含，嘻嘻

    知识点如下：

    1. 安装Sublime；
    2. 安装Package Control(必须)，并通过插件汉化；
    3. 无法输入中文问题；
    4. Sublime插件，打造python专属编辑器

    安装Sublime

    1. 安装GPG密钥：

      wget -qO - https://download.sublimetext.com/sublimehq-pub.gpg | sudo apt-key add -

    2. 确保apt已设置为使用https源：

      sudo apt-get install apt-transport-https

    3. 选择要使用的频道：

      稳定
      echo “deb https://download.sublimetext.com/ apt/stable/“ | sudo tee /etc/apt/sources.list.d/sublime-text.list
      开发
      echo “deb https://download.sublimetext.com/ apt/dev/“ | sudo tee /etc/apt/sources.list.d/sublime-text.list

    4. 更新apt源并安装Sublime Text

      sudo apt-get update
      sudo apt-get install sublime-text

    英文不好嘛？那就汉化一下喽！

    1. Sublime Text安装Package Control
      快捷键ctrl+` ， 然后粘贴如下代码到控制台并回车：
      如无异常，重启一下就ok了……

      import urllib.request,os,hashlib; h = ‘6f4c264a24d933ce70df5dedcf1dcaee’ + ‘ebe013ee18cced0ef93d5f746d80ef60’; pf = ‘Package Control.sublime-package’; ipp = sublime.installed_packages_path(); urllib.request.install_opener( urllib.request.build_opener( urllib.request.ProxyHandler()) ); by = urllib.request.urlopen( ‘http://packagecontrol.io/' + pf.replace(‘ ‘, ‘%20’)).read(); dh = hashlib.sha256(by).hexdigest(); print(‘Error validating download (got %s instead of %s), please try manual install’ % (dh, h)) if dh != h else open(os.path.join( ipp, pf), ‘wb’ ).write(by)

    2. 点击Preferences下的Package Control后选择install package选项
      期间可能会跳出一个弹窗，点确认就好.

    3. 然后Ctrl+Shift+P调出Install Package,
      搜索ChineseLocalization选项回车确认静等完成，然后重启Sublime就好了



    无法在输入中文？

    1. git clone https://github.com/lyfeyaj/sublime-text-imfix.git
    2. cd sublime-text-imfix
    3. ./sublime-imfix
      !!! 完成，重新打开你的sublime窗口，切换中文输入法，就可以输入中文了。

      安装插件，打造python专属编辑器

      通过插件，我们可以很好的提高开发效率，所以我在此例举几个非常不错的插件
      安装方法么？
      Ctrl+Shift+P调出Install Package,
      插件如下：
    4. SideBarEnhancements: 扩展了侧边栏中菜单选项的数量，从而提升你的工作效率。
    5. SublimeTmpl：新建文件模板插件，可以支持多种语言例如Python、PHP等，在settings-user中设置上自己的信息。
    6. Terminal：打开一个命令窗口，用于各种命令操作
    7. AutoPep8：python开发规范pep8
    8. Anaconda：自动匹配关键字等实用功能，有效提高开发效率
    9. SublimeREPL：直接运行当前文件，可以方便调试，与Terminal可联合使用。
